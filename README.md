# smart-contract-management-system

## Программная система для управления смарт-контрактами цифровых активов

### Функциональность:
- Визуальный конструктор цифровых активов
- Генерация кода смарт-контрактов на Solidity
- Статический анализ кода (Slither + Mythril)
- Развертывание (деплой) в блокчейн-сети
- Электронный реестр смарт-контрактов

### Технологии:
- Backend: Node.js + NestJS
- Frontend: React + Next.js
- База данных: PostgreSQL
- Блокчейн: Web3.js / Ethers.js
- Контроль версий: Git + GitHub
